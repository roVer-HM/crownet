# Supermarket - Vadere-only

This folder contains Vadere pedestrian simulation scenarios for modeling pedestrian dynamics in an indoor supermarket environment. There are no OMNeT++ network simulation components.

## Vadere Scenario Details

- **Topography**: 52.5m × 32.5m (indoor supermarket layout with shelves and aisles)
- **Simulation Time**: 1000s
- **2 Sources**: Negative exponential distribution (mean 16s), targeting different checkout areas
- **Pedestrian Speed**: mean 0.6 m/s
- **Psychology Layer**: Enabled (CooperativeCognitionModel)
